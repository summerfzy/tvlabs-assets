# tvlabs-assets

Public brand image assets for TV Labs email/comms (served via jsDelivr CDN).
Contains **only** static brand artwork — no application code, no secrets.

Email clients (Gmail in particular) only render hosted `https` images, so the
TVLabs logo and email section icons live here and are referenced by absolute URL.

## Files
- `tvlabs-icon-purple.png` — TV logo mark, vivid `#4C2EFF`, on `#101828` (dark email bg)
- `ic-calendar.png`, `ic-plane.png`, `ic-card.png`, `ic-gift.png` — section icon tiles

## Usage (jsDelivr, pinned to a tag for immutability)
```
https://cdn.jsdelivr.net/gh/summerfzy/tvlabs-assets@v1/tvlabs-icon-purple.png
```
